My Plants

3rd semester group exam project

  • Web Application concept developed with HTML, CSS and JavaScript
  • Single Page Application
  • Firebase for login authentification and data storage

"My Plants" gives the user an overview over their plants at home and will give guidance on how to take care of those.
With Firebase we made it possible to make a personal profile, that storages user data.
In the plant library the user can search and (in the future) scan the plants they have at home wich they can add to their own "My Plants"-profile. Here the user gets information about each plant, that tells when each plant needs to be watered and how much water to give each plant.
In the future we would like to develop the calendar and notifications functions.

Klimaspillet

2nd semester group project

  • Web Application
  • HTML, CSS and JavaScript
  • Interaction
  • UX
  • Usability
  • SoMe - Marketing strategy plan

Interactive Web Application developed for the Greenhouses in the Botanical Gardens in Aarhus. We created an interactive learning platform with a quiz called Klimaspillet that follows a specific route through the Greenhouses. The users of Klimaspillet are children in the 6th grade visiting the Greenhouses. The purpose with the web app is to make the users interact and learn about sustainability at the same time.
